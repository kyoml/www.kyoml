# Booleans

Booleans are all lowercase. Supported values are
- `true`
- `yes`
- `false`
- `no`

`yes` and `no` are simple aliases for `true` and `false`

<pre class="ace-editor" lang="kyoml">

field1 = true
field2 = yes
field3 = false
field4 = no
</pre>
